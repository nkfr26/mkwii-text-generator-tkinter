■ Description
Characters that can be used are A-Z 1-9 : . + - /
Second font are available for 1-9 and - /
They can be used by enclosing them in double quotation marks (").
Also, space enclosed in " is one-fourth length of normal space.
Fine adjustment can be done with <>. (> x 5 = " ")

Example
time 00:00.000
lap " 1"  00:00.000
"km/h" sp<<<<<eed
